Как создать файл определенного размера в Windows?

Эта команда создаст в корне диска C:\ новый файл с именем ’10mb-file.txt’, размером 10000000 байт (10 Мб)

fsutil file createnew C:\10mb-file.txt 10000000
Как создать файл определенного размера в Linux?

Команда fallocate быстро создаёт файл определённого размера, выделяя место на диске. Создать файл размером 1 ГБ с помощью fallocate:

fallocate -l 1G file1gb

После создания файла вы можете проверить его размер с помощью команды ls:

ls -lh filename
Как удалить все разделы на HDD(SSD) в Windows?

Запускаем терминал и выполняем:

diskpart
list disk

Выбираем необходимый диск:

select disk 2
clean
list partition
Как сделать, чтобы apt не ругалась на битые зависимости у пакета?

Посмотри на nano /var/lib/dpkg/status и заставь его думать, что у пакета нет тех зависимостей.

Как сгенерировать SPKI для сертификата?

Используйте эту команду для генерации SPKI

echo | openssl s_client -connect 192.168.1.21:853 2>/dev/null | openssl x509 -pubkey -noout | openssl pkey -pubin -outform der | openssl dgst -sha256 -binary | openssl enc -base64
Postfix: NIS domain name not set

Postfix сыпет в логи:

dict_nis_init: NIS domain name not set - NIS lookups disabled

Выполним команду:

postconf | grep nis

Примерный результат:

alias_maps = hash:/etc/aliases, nis:mail.aliases

Выполним:

postconf -e alias_maps=hash:/etc/aliases

И смотрим:

postconf | grep nis

Должен быть пустой результат, перезагружаем postfix и радуемся:

systemctl restart postfix
Permission denied to github-actions bot

Любопытный случай отказа в доступе к репозиторию в рабочем процессе Github Actions
При указание токена PAT непосредственно в команде git push, завершается с ошибкой отказа в доступе: The requested URL returned error: 403.

Решение

Указание в действие токена PAT с доступом на чтение и запись к нужным репозиториям исходного кода:

      - name: Checkout repository
        uses: actions/checkout@v4
        with:
          ref: gh-pages
          token: ${{ secrets.SECRET_PAT }}

Теперь нет нужды отдельно указывать токен для git push. Конфигурация git остается неизменной на всех этапах.
