Отправка почты в php через msmtp

08.04.2011Меньше 1 минутыphp

В этой заметке я рассмотрю отправку почты в php через msmtp, в качестве smtp сервера я буду использовать сервер Яндекса.

Прежде всего устанавливаем msmtp:

apt install msmtp

В MSMTP файл конфигурации может быть глобальным или пользовательским, глобальный находится в /etc/msmtprc а пользовательский в домашней директори пользователя .msmtprc точка впереди не случайно, файл является скрытым.

Я буду использовать глобальный файл конфигурации, создаем и настраиваем:

nano /etc/msmtprc

Содержимое должно быть такое:

account yandex
host smtp.yandex.ru
port 587
protocol smtp
auth on
from <ваш email>
user <ваш email>
password <ваш пароль>
tls on
tls_starttls on
tls_certcheck off
account default : yandex

Теперь правим php.ini

nano /etc/php5/apache2/php.ini

Находим sendmail_path = и заменяем на:

sendmail_path = /usr/bin/msmtp -t -i -a yandex

Перезапускаем apache:

/etc/init.d/apache2 restart

Все, теперь можно отправлять почту.
