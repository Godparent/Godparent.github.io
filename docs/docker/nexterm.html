Nexterm

20.12.2024Меньше 1 минутыDockerterminalssh

Nexterm — веб-терминал с открытым исходным кодом, который позволяет:

  • Подключайтесь удаленно через SSH, VNC и RDP
  • Управление файлами через SFTP
  • Развертывание приложений через Docker
  • Управление контейнерами Proxmox LXC и QEMU
  • Безопасный доступ с двухфакторной аутентификацией

Установить Nexterm с помощью Docker

services:
  nexterm:
    ports:
      - "6989:6989"
    restart: always
    volumes:
      - /root/docker/nexterm:/app/data
    image: germannewsmaker/nexterm:1.0.2-OPEN-PREVIEW
volumes:
  nexterm:
